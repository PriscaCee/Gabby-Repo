# PrimeHomes – Real Estate Landing Page

*Live Website →* https://priscacee.github.io/Gabby-Repo/

A clean, modern, fully responsive real estate website built in *one single HTML file* — no frameworks, no build tools, just pure code!

### Features
- Hero section with CTA
- Featured properties with hover effects
- Responsive grid (works perfectly on mobile)
- Fixed navigation bar
- Beautiful typography & spacing

### Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, CSS variables)
- Google Fonts & Font Awesome (via CDN)
- Git & GitHub
- Deployed with *GitHub Pages*

### Project Status
- Fully deployed and live
- First ever website I pushed to GitHub from scratch!
